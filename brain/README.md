# 🧠 This folder is your brain

Every `.md` file in here is one memory. Your AIs **read** these files before they
answer, and **write** new ones when they learn something durable about you. Then
`sync_brain.py` mirrors this whole folder to your Supabase database and back — so
a fact Claude learns shows up for Codex, and vice versa.

**You don't have to write these by hand.** Just work with your AI normally; it
adds and updates files here as it goes. But you can edit or add them yourself
too — it's plain text you own.

Format is simple: a short title, then the fact. See `example-about-me.md`.
Delete that example once you've got your own memories in here.
